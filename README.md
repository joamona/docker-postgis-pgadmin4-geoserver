### Instructions

- Clone the repo.
- Read the *.yml of each application  and follow the instructions.
- To start the applications separatelly, execute the following en each folder:

    docker-compose up 

- To start all aplications, execute:
    
    ./start_all.sh

- To stop all applications, execute:

    ./stop_all.sh
